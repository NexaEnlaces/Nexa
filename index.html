Credenciales Clientes
Claves SII · Previred · Representantes Legales · Solo visible para ti
🔒 Cómo funciona
Las credenciales se guardan en un Google Sheet separado y privado, distinto al Sheet compartido con tu socia. Solo tú puedes verlas cuando estás conectada con Google. Tu socia ve la ficha del cliente pero no las claves.
⚙️ Configuración necesaria
1. Crea un Google Sheet nuevo (solo para ti, sin compartir)
2. Copia su ID en Configuraciones → Google Sheets → Spreadsheet ID Credenciales
3. Conecta con Google y usa "Subir a Sheet"

Sincronización en tiempo real con tu socia
⚙️ Paso 1 — Configurar credenciales
Google Cloud Console → APIs → Credenciales → OAuth 2.0 Client ID
URL de tu Google Sheet: docs.google.com/spreadsheets/d/ESTE_ID/edit
🔒 Sheet Privado de Credenciales — separado del Sheet principal. Solo visible para ti.
Crea un Sheet separado, solo compartido contigo. Guarda aquí las claves SII, Previred, etc.
🔐 Paso 2 — Conectar con Google
Al hacer click, Google pedirá que autorices el acceso a tu hoja de cálculo. Solo necesitas hacerlo una vez por equipo. La autorización expira cada hora y se renueva automáticamente.
🔄 Paso 3 — Sincronizar datos
Primera vez: usa "Subir todo a Sheets" para crear las hojas y cargar tus datos.
Uso diario: usa "Sincronizar" para enviar los cambios locales, o "Cargar desde Sheets" para recibir los de tu socia.
