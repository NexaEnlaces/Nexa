Google Sheets
Sincronización en tiempo real con tu socia
⚙️ Paso 1 — Configurar credenciales
Google Cloud Console → APIs → Credenciales → OAuth 2.0 Client ID
URL de tu Google Sheet: docs.google.com/spreadsheets/d/ESTE_ID/edit
🔐 Paso 2 — Conectar con Google
Al hacer click, Google pedirá que autorices el acceso a tu hoja de cálculo. Solo necesitas hacerlo una vez por equipo. La autorización expira cada hora y se renueva automáticamente.
🔄 Paso 3 — Sincronizar datos
Primera vez: usa "Subir todo a Sheets" para crear las hojas y cargar tus datos.
Uso diario: usa "Sincronizar" para enviar los cambios locales, o "Cargar desde Sheets" para recibir los de tu socia.
